# 📊 Sales Analytics Dashboard

## Overview
The Sales Analytics Dashboard is an interactive Power BI project developed to analyze and visualize sales data effectively. It provides a comprehensive view of key sales metrics, customer behavior, product performance, and regional trends through interactive visualizations and KPI tracking.

## Key Features
- Total Sales, Quantity, Average Sales, and Profit KPIs
- Product Category Performance Analysis
- Customer Type Analysis (New vs Returning Customers)
- Payment Method Distribution
- Sales Channel Comparison
- Regional Sales Trends and Performance
- Interactive Filters and Dynamic Visualizations

## 🛠️ Tools & Technologies
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Data Visualization

## 📈 Insights Covered
This dashboard helps analyze:
- Overall sales performance
- Customer purchasing behavior
- Product category trends
- Payment method preferences
- Sales channel effectiveness
- Regional sales distribution

## 🎯 Project Purpose
This project was created to strengthen my Power BI, data analysis, and data visualization skills. It demonstrates my ability to transform raw sales data into meaningful insights through an interactive and professional dashboard.
